2. run

```

```



실행중인 컨테이너 중지

```
sudo docker rm $(docker ps -a -q)
```

모든 이미지 삭제

```
docker rmi $(docker images -q) --force
```

도커 이미지, 컨테이너, 볼륨, 네트워크 등 일괄 삭제

```
sudo docker system prune
```
